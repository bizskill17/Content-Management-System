# Role of GitHub in JhatPatAI

GitHub is the central hub for source code, version control, and automated deployment.

## 🛠️ Key Responsibilities

- **Source Code Hosting:** Securely store PHP, Next.js, and SQL files.
- **Version Control:** Track every change and revert to previous versions if bugs occur.
- **Continuous Deployment (CI/CD):** Automate updates to Hostinger or Vercel whenever code is "pushed" to the repository.
- **Team Collaboration:** Enable multiple developers to work on different features (e.g., Auth, UI, Sync Engine) without conflicts.
- **Issue Tracking:** Manage bugs, feature requests, and the product roadmap via GitHub Issues.

## 🔄 Integration Workflow
**GitHub (Code Source)** → **Vercel/Hostinger (Automatic Build/Deploy)** → **User (Live Site)**
